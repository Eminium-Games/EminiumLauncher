Bienvenue sur Eminium Launcher

Découvrez comment utiliser votre nouveau launcher

1

Connectez-vous

Utilisez votre compte Eminium ou Croissant-API.fr pour vous connecter. Cliquez sur l'un des boutons de connexion pour commencer.

2

Configurez vos paramètres

Dans l'onglet "Jouer", ajustez la mémoire allouée, la distance de rendu et les FPS selon les performances de votre ordinateur.

3

Lancez le jeu

Cliquez sur "Installer / Mettre à jour" pour vérifier les fichiers, puis sur "Jouer" pour lancer votre aventure Minecraft.
